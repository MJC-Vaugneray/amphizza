## Docker

### Build local

```bash
docker build -t amphizza .
docker run --rm -p 8080:8080 --name amphizza amphizza
```
### J'veux juste l'image

```bash
docker run --rm -p 8080:8080 --name amphizza mjcvaugneray/amphizza
```
